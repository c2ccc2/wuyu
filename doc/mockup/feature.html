# 功能介绍

# 业务模块

1、原型管理：创建原型、查看发布、编辑原型。

2、发布管理：创建版本发布，查看原型文件、管理分享权限、删除发布。

3、审核管理：审核发布内容、查看发布内容。

# 原型浏览

1、查看原型链接：通过发布的原型链接，查看原型的发布版本lieb

2、查看原型：直接通过url方式查看所发布的原型。

# 管理模块

1、审批管理：管理是否开启审批，默认不开启。

2、角色管理：管理系统角色

3、组织管理：用户管理、角色管理、重置密码、批量删除。

4、职位管理：包括筛选、新增、修改、删除、批量删除等功能。
