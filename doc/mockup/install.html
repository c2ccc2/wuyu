    # 安装部署

    # 1. windows

    • 下载deploy文件夹，找到windows 目录

    • 安装目录下的readme进行数据库部署

    • 运行目录下server.exe

    • 系统即开始运行

    • 登录使用

    # 2. linux

    推荐使用compose方式，该脚本已经打包好mysql、nginx以及后端程序

    • 下载deploy文件夹，找到compose 目录

    • 进入compose 目录

    • 运行 docker compose up (或者 docker compose up -d)

    • 系统即开始运行

    • 登录使用
