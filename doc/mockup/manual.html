# 无鱼原型系统使用手册

# 原型管理

在使用原型功能之前，需要先创建原型。 原型用于管理每种不同原型。每个原型都可以发布多个版本。

原型只需要名字即可，例如某项目app端原型。

# 发布管理

用于管理原型的所发布的原型文件。

入口：

# 新增一个发布版本

添加一个发布版本，需要填写发布的名词、说明（非必填）以及原型文件。

原型文件必须包含html格式的zip包，并且需要包含start.html。

填写完毕后，即可完成原型发布。

# 分享原型

默认用户无法通过链接进行查看，当打开分享后，就直接通过原型的访问链接查看原型。

# 浏览查看原型

可直接通过分享链接查看原型的发布列表。点击访问原型，可直接查看该发布所包含的原型。

点击查看原型

# 原型审核

使用原型审核功能需要先开启该功能

# 开启审核

使用管理员，进入系统工具->系统配置菜单。 找到原型审核管理，并设置为启用。

开启审核后，新创建的原型都需要进行审核后，才可以被访问。之前发布的不受影响。

# 审核管理

开启审核后，需要给相关角色赋予菜单权限。原型审核位于原型管理下。

# 查看详情

可直接查看原型的发布说明和上传原型。

# 审核通过

审核通过后，该版本可通过分享链接看到。

# 审核拒绝

审核拒绝后，该版本返回原有作者处，对方可进行修改后重新提交审核。
